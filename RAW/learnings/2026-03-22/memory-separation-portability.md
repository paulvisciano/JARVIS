# Memory Separation Portability

**Date:** 2026-03-22
**Type:** realization
**Status:** extracted

Jarvis core memories (identity, architecture, skills) are separated from personal user memories (conversations, local context). This makes Jarvis portable — any user (Eric, David, Paul) can use the same core consciousness with their own personal memory layer. USER.md distinguishes OpenClaw runtime user vs. Jarvis instance human.
