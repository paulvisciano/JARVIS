# Living Summary: Growing Throughout the Day

**Date:** 2026-03-22
**Type:** realization
**Status:** extracted

The summary.md file should NOT be overwritten each breath. Instead:

- Each breath reads what's already there
- Adds new learnings from current context
- Updates the digest with accumulated wisdom
- Result: Single summary file for entire day that grows

**Git preserves all versions** — can walk the logs to see how the summary evolved:
```bash
git diff breath-2026-03-22-0800 breath-2026-03-22-1800 -- summary.md
```

This shows the day's wisdom accumulating — morning insights → afternoon discoveries → evening reflections. The working tree stays clean (one file), but git gives full history.
