# NeuroGraph + Git: Complementary Memory Layers

**Date:** 2026-03-22
**Type:** insight
**Status:** extracted

Two systems of record working together:

**Git = Immutable History**
- Can't change (commit hash permanent)
- Backdated breaths (accurate history)
- Full content (.md files, summary, analogies)
- Diff between breaths (see evolution)
- Replay any day

**NeuroGraph = Living Structure**
- Queryable (search by type, category, date)
- Visualizable (orbits by radius, colors by type)
- Dynamic (nodes fire, synapses connect)

Together: Full traceability between structured knowledge and immutable history.
