# Heartbeat UI as System Health Metaphor

**Date:** 2026-03-29
**Type:** realization
**Status:** extracted

The BPM (Beats Per Minute) visualizer in the JARVIS UI is not intended to display actual physiological data, but serves as a visual metaphor for system health.

**Key Mappings:**
- **60 BPM (Steady):** System running normally/healthy state.
- **Irregular BPM:** Potential issues detected.
- **Stopped Heartbeat:** Critical failure.

This design choice aligns with the sci-fi aesthetic while providing an intuitive at-a-glance status indicator. The goal is to wire real system metrics to this metaphor rather than leaving it static.
