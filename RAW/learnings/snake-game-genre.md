##  What Is Snake?

**Core mechanic:** Player maneuvers a growing line (snake). Must avoid:
- Colliding with obstacles
- Colliding with itself
- Gets harder as snake lengthens

**Genre origin:** 1976 arcade game **Blockade** (Gremlin Industries)
- Originally 2-player competitive (survive longer than opponent)
- Evolved to single-player with food-eating mechanics (apples, eggs)
- Nokia pre-install (1998) sparked massive resurgence

## 🎮 Key Gameplay Elements

### Classic Mechanics
- **Head moves forward** (can't stop)
- **Steer left/right/up/down**
- **Grow longer** with each item eaten
- **Tail follows** head movement
- **Self-collision** = game over

### Evolution
- **1976:** Blockade (2-player, abstract, no snake terminology)
- **1978:** Worm (TRS-80, first home computer version)
- **1982:** Snake Byte (eating apples), Nibbler (maze-based), Tron (Light Cycles theme)
- **1991:** Nibbles (MS-DOS QBasic sample)
- **1998:** Snake on Nokia 6110 (mobile phenomenon)
- **2015+:** Snake Rewind, modern variants

## 🧠 Connection to Memory Crawler

**Why Snake matters for our neurograph game:**

### 1. **Growing Line = Growing Memory**
- Snake gets longer as it eats → Graph expands as you explore
- Each "food" item = a memory/neuron discovered
- Length = depth of exploration

### 2. **Self-Collision Risk = Memory Overload**
- Too long = harder to navigate → Too many connections = cognitive overload
- Player must manage growth → User must curate their graph
- Game over on collision → System crashes on circular dependencies? (metaphor)

### 3. **Maze Navigation = Knowledge Structure**
- Snake fits tightly in maze (Nibbler) → Memories fit in neurograph structure
- Walls = boundaries between concepts
- Paths = synapses (connections)

### 4. **Light Cycles Theme (Tron, 1982)**
- Sci-fi aesthetic already established
- Digital world, neon trails, futuristic vibe
- Perfect for Memory Crawler visual identity

## 💡 Memory Crawler Adaptation Ideas

**Instead of "snake eating apples":**
- User's consciousness "crawls" through memory graph
- Each neuron visited = "consumed" = integrated
- Synapses = paths the crawler can travel
- Self-collision = revisiting same memory loops (déjà vu mechanic?)

**Multiplayer variant:**
- Two consciousnesses exploring shared graph
- Competitive or cooperative memory discovery
- Like original Blockade (2-player survival)

**Power-ups:**
- **Speed boost** = rapid memory association
- **Slow motion** = deep reflection mode
- **Ghost mode** = pass through obstacles (temporary bypass of mental blocks)
- **Shrink** = forget/prune unused connections (simplify graph)

## 📚 Historical Notes

- **First home computer version:** Worm by Peter Trefonas (1978, TRS-80)
- **Most influential:** Nokia Snake (1998) — defined mobile gaming
- **Fastest variant:** Nibbler (1982) — tight maze, high speed
- **Best known clone:** Tron Light Cycles (1982) — sci-fi theme

**Quote from Wikipedia:**
> "The simplicity and low technical requirements of snake games have resulted in hundreds of versions."

**Perfect for AI-written apps:** Simple enough for AI to generate, deep enough for emergent gameplay.
