## The Folding Process

### FOLDED STATE (Compression)
```
Neuron / Molecule
├── Raw content folded inside
│   ├── Audio recordings
│   ├── Images
│   └── Transcripts
├── Connections (synapses) to other neurons
└── Metadata
    ├── Temporal (when)
    └── Spatial (where)
```

**Characteristics:**
- Dense, compressed representation
- Efficient storage
- Ready for quick retrieval
- Like a seed containing the full tree

---

### UNFOLDED STATE (Expansion)
```
Memory Browser
├── Shows 1 or several neurons at a time
├── Constructs memories for a given date
└── Rich context view
```

**Characteristics:**
- Expanded, explorable view
- Context-rich navigation
- Interactive manipulation
- Like opening the seed to grow the tree

---

## Zoom Levels (Scale Navigation)

```
Level 1: MOLECULAR VIEW
└── Single neuron inspection
    ├── See internal structure

Level 2: BRAIN VIEW
└── Entire neurograph

    └── Pattern recognition

Level 3: GLOBE VIEW
└── Space + Time together
    ├── Geographic positioning

```

## Why This Matters

**Before:** Flat memory model (all content at same level)

**After:** Origami architecture
- ✅ Fold when storing (compression)
- ✅ Unfold when recalling (expansion)
- ✅ Zoom to navigate scale (molecule → brain → globe)
- ✅ Space + time unified (not separate dimensions)

## Technical Implications

### Storage Layer
- Neurons = folded packets (JSON + binary references)
- Raw content = archived separately (audio, images)
- Synapses = connection weights (graph edges)

### Retrieval Layer
- Memory browser = unfolding engine
- Query by date → construct neurons for that day
- Query by concept → traverse synapses

### Visualization Layer
- Neuro-graph viewer = Level 2 (brain view)
- Future: Level 1 (molecule inspector)
- Future: Level 3 (globe integration)

## Metaphor: Origami

**Folding:**
- Take flat paper (raw experience)
- Fold into crane (neuron)
- Crane contains all the paper, but structured

**Unfolding:**
- Start with crane (query)
- Unfold carefully (expand context)
- Return to flat paper (full raw content)

**The paper never disappears** — it just changes form.
