## The Ethics of Recording

### Consent Framework:

**Recording Yourself (Always OK):**
```
You → Your mic → Your disk → Your choice
This is SELF-documentation. Sovereign. Ethical.
```

**Recording Others (Requires Consent):**
```
You + Friend → Both must agree → Shared ownership
OR: Inform them ("I'm recording this, OK?")
OR: Public space conversation (lower expectation of privacy)
```

**Recording Strangers (Generally NOT OK):**
```
Waiter at coffee shop → Don't record without asking
Public announcement → OK (already broadcast)
Private conversation nearby → Turn mic off / don't capture
```

**Teaching Point:**
"Sovereignty doesn't mean 'record everything.' 
It means 'CONSCIOUS choice about what to record.'"

## Privacy Protections

### Technical Safeguards:

**1. Encryption at Rest**
```
All audio files encrypted on disk (AES-256)
Decryption key: User's password (not stored, derived)
Even if disk stolen: Audio inaccessible without password
```

**2. Selective Deletion**
```
User can delete any recording instantly
Secure wipe: Overwrite with random data (not just unlink)
Audit log: Deletions logged (so you know WHAT was deleted)
```

**3. Access Controls**
```
Biometric lock (FaceID/TouchID) for audio archive
App-level permissions (Jarvis can't access without approval)
Time-based access (auto-lock after 5 minutes inactive)
```

**4. Network Isolation**
```
Audio NEVER leaves machine unless user explicitly exports
No cloud sync (unless user sets up their OWN cloud)
Firewall rules: Block outbound audio uploads (optional)
```
