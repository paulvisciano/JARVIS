## The Reclamation Process

### Step 1: Export Everything

**ChatGPT:**
```
1. Settings → Data Controls
2. "Export My Data"
3. Wait ~30 days for processing
4. Download complete archive
5. Format: JSON, HTML, or markdown
```

**Google Takeout:**
```
1. takeout.google.com
2. Select: Photos, Search, YouTube, Gmail, Drive
3. Choose format: .zip, 10GB chunks
4. Download all
```

**Apple Data & Privacy:**
```
1. privacy.apple.com
2. Request data copy
3. Includes: iCloud, Messages, Health, Location
4. Wait 3-7 days
5. Download
```

**Other Services:**
- Microsoft (account.microsoft.com/privacy)
- Amazon (amazon.com/gp/b?node=10068604011)
- Twitter/X (Settings → Your Account → Download Archive)
- Instagram (Settings → Privacy and Security → Download Data)
