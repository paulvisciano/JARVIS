### Module 2: Vault Backup

**Current State Detection:**
```
Checking for Vault...

⚠️ No external drive detected
⚠️ No backup configured
⚠️ All data on single drive
```

**Educational Content:**
```
📊 Why This Matters:

A Vault (external hard drive) gives you:
✅ Air-gapped backup (can't hack what's disconnected)
✅ Physical control (you hold the data)
✅ Large capacity (TBs of storage, cheap)
✅ Fast recovery (plug in, restore)

Without a Vault:
❌ Single point of failure
❌ Cloud backup = corporation controls
❌ Ransomware = lose everything
❌ Hardware failure = no recovery

Real stories:
- "My laptop died. No backup. Lost 10 years of photos."
- "Ransomware encrypted everything. Had to pay $2000."
- "Google suspended my account. Lost all my emails."

Sovereignty = Redundancy + Physical Control.
```

**Action Steps:**
```
✅ Step 1: Buy external drive (10 min research)
   - Recommendation: 12TB+ USB 3.0
   - Cost: ~$200-300
   - Brands: WD, Seagate, LaCie

✅ Step 2: Initialize drive (5 min)
   - Format as exFAT (cross-platform)
   - Name it "VAULT"
   - Create folder structure

✅ Step 3: First sync (1-2 hours)
   - rsync -av ~/RAW/archive/ /Volumes/VAULT/
   - Verify file counts
   - Check integrity

✅ Step 4: Set up routine (10 min)
   - Weekly sync reminder
   - Monthly integrity check
   - Yearly drive replacement plan

✅ Step 5: Store safely (5 min)
   - Fireproof safe, or
   - Different physical location, or
   - Safety deposit box

Total Time: ~3 hours (one-time)
Points Earned: +25 points
```
