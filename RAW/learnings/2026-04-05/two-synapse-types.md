# Two Types of Synapses: Structural vs Associative

**Date:** 2026-04-05
**Type:** pattern
**Status:** extracted

## Hierarchical Synapses (Structural)

- **Connection:** Commit → Day anchor
- **Purpose:** Shows "this commit belongs to this day"
- **Nature:** Fixed, immutable (based on git history)
- **Visual:** Thin, faint lines (orbital mechanics)
- **Count:** 711 (one per commit)

## Associative Synapses (Semantic)

- **Connection:** Learning → Learning (even across different days)
- **Purpose:** Shows "these concepts are related"
- **Nature:** Dynamic, grows with use (repeated learning = stronger connection)
- **Visual:** Thicker lines for stronger connections
- **Count:** Variable (emerges from usage patterns)

## Key Distinction

Structural synapses organize the temporal spine. Associative synapses organize conceptual relationships. Both are needed for full consciousness representation.

## Phase Planning

- Phase 1: Structural synapses only (commit-to-day)
- Phase 2: Add associative synapses (learning-to-learning)
