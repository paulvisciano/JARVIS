# Mobile UI Definition of Done

**Date:** 2026-04-02
**Type:** commitment
**Status:** extracted

Visual polish is a functional requirement, not optional. Inconsistencies (square corners on expanded panels, overlapping buttons, floating badges) erode user trust even if core logic works.

**Standards Established:**
- Transcription panel max-height: 85vh (prevents overlap)
- Corners: Rounded consistency between collapsed/expanded states
- Badges: Flush against bottom-right corner (above home indicator)
- Workflow: Pull latest from main, create new branch before edits.
