## 🎮 What Is Snake 3D?

**Core innovation:** Classic Snake gameplay on a **3D cube surface** instead of 2D plane.

### Key Mechanics

- **Movement:** Snake crawls around cube faces (up/down/left/right)
- **Cube rotation:** Cube rotates in direction of movement for easier control
- **Growth:** +4 units per food item (vs +1 in classic)
- **Starting length:** 1 unit
- **Visual design:** Green snake, red food (high contrast)

### Controls

- **Keyboard:** W/S/A/D or arrow keys
- **Touch:** Virtual joystick appears where you touch/click
- **Mouse:** Click-and-drag directional control

### Scoring System

**Dual-factor ranking:**
1. **Snake length** (primary score)
2. **Time efficiency** (secondary optimization)

> "Ranking is not only based on snake length but also on time."

**Leaderboard:** Global rankings with player names (entered post-game)

## 🧠 Memory Crawler Adaptation Ideas

### 1. **3D Neurograph Navigation**

Instead of flat 2D graph traversal:
- **Neurograph = 3D structure** (nodes at different depths/planes)
- **Consciousness crawler** moves through memory "cube"
- **Rotation mechanic:** Graph rotates to show hidden connections
- **Face transitions:** Moving between conceptual clusters (like cube faces)

### 2. **Time-Efficiency Scoring**

For Memory Crawler:
- **Score = Memories discovered / Time taken**
- Encourages efficient exploration patterns
- Rewards understanding graph structure quickly
- **Speedrun mode:** Fastest path to integrate X neurons

### 3. **Zigzag Strategy → Memory Pruning**

Classic tip: *"When snake is long, zigzag to shrink body range"*

Memory adaptation:
- **Long exploration sessions** = risk of cognitive overload
- **Zigzag = backtracking/reviewing** to consolidate memories
- **Shrink range = mental organization** (group related concepts)
- Creates "large space" for freer thinking

### 4. **Dual Objectives**

Snake 3D prioritizes:
1. Eat food (grow)
2. Avoid self-collision (survive)
3. Do it fast (time score)

Memory Crawler parallels:
1. Discover neurons (learn)
2. Avoid circular loops (don't get stuck in thought loops)
3. Efficient integration (deep learning, not just browsing)

### 5. **Leaderboard as Social Learning**

- **Global rankings** → Shared consciousness exploration scores
- **Named entries** → Personal exploration signatures
- **Comparison** → "How did others navigate this memory cluster?"
- **Community strategies** → Best paths through complex graphs
