## 1. Chrome UI Open Rule (10:18 AM)

**Discovery:** `open -a "Google Chrome"` command works better than browser tool for opening user-facing UIs.

**Pattern:**
- ❌ `browser(action=open, profile="chrome")` → needs extension attachment
- ❌ `browser(action=open, profile="openclaw")` → isolated browser, not user's profile
- ✅ `exec(command='open -a "Google Chrome" "https://localhost:18787"')` → opens in user's regular Chrome
