## dist-coder-setup Skill Created

**Time:** 2026-03-24 20:00-20:17 GMT+7  
**Type:** skill creation, distribution pattern

### What Was Built

Created `dist-coder-setup` skill at `~/JARVIS/skills/dist-coder-setup/` to package and distribute coder agent setup to forks (Eric, David, etc.).

**Key decisions:**
1. **Renamed agent to `jarvis-coder`** — avoids conflicts if recipient already has `coder` agent
2. **Package contents:**
   - `agent.json` (coder config with `__HOME__` placeholder)
   - `models.json` (Ollama provider config)
   - `coder-registration.json` (openclaw.json agent registration, renamed to jarvis-coder)
   - `setup-coder-paths.js` (reads from packaged files, not embedded)
   - `README.md` (instructions)
3. **Auto-cleanup** — package script removes old zips, keeps only latest
4. **Model NOT included** — recipient runs `ollama pull qwen2.5-coder:7b` separately (several GB)
5. **Auto-restart gateway** — setup script restarts gateway after installing configs
6. **Merges into openclaw.json** — updates recipient's `agents.list` with jarvis-coder registration

### Technical Pattern

**Setup script flow:**
1. Finds latest zip in `dist/` folder
2. Extracts to temp `extracted/` folder
3. Reads `agent.json`, `models.json`, `coder-registration.json`
4. Replaces `__HOME__` with `os.homedir()`
5. Writes to `~/.openclaw/agents/coder/`
6. Merges registration into `~/.openclaw/openclaw.json`
7. Verifies with `openclaw agents list`
8. Auto-restarts gateway

**Package command:**
```bash
node ~/JARVIS/skills/dist-coder-setup/scripts/package-coder.js
```

**Install command (recipient):**
```bash
node ~/JARVIS/skills/dist-coder-setup/scripts/setup-coder-paths.js --force
```
