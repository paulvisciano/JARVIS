## UI Integration Idea (Paul's Vision)

**Concept:** Video as ORB background
```
┌─────────────────────────┐
│                         │
│    ┌─────────────┐      │
│    │   VIDEO     │      │
│    │    ORB      │      │
│    │  (looping)  │      │
│    │             │      │
│    └─────────────┘      │
│                         │
│  [Click to engage]      │
│                         │
└─────────────────────────┘
```

**Behavior:**
1. **Idle state:** ORB shows looping video (neural network pulsing)
2. **On click:** Video "engages" — network fires faster, brighter
3. **During interaction:** ORB reacts to user input (typing, queries)
4. **After interaction:** Returns to idle pulse

**Technical Implementation:**
- **HTML5 `<video>` element** with `loop` + `muted` attributes
- **CSS circular mask** (`border-radius: 50%`)
- **Click handler** → add "engaged" class (speed up playback, increase brightness)
- **Optional:** WebAudio API for reactive sound (if desired)
